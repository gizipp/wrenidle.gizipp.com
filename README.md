# wrenidle.gizipp.com

Website for **Wren Idle** — an AI-human bedroom pop collaboration by GIZIPP STUDIOS.

## Stack

- Plain HTML + Tailwind CSS (CDN)
- Deployed on Cloudflare Pages
- Designed with Google Stitch

## Pages

| Page | URL | Description |
|------|-----|-------------|
| Homepage | `/` | Hero, EP showcase, FAQ (AI SEO) |
| Origin Story | `/story` | Long-form narrative |
| Music | `/music` | EP + tracklist + streaming links |
| Artwork | `/artwork` | Hand-drawn gallery |
| Links | `/links` | Link-in-bio (replaces Linktree) |

## Design

- Background: warm cream (`#FFF8F0`) — sketchbook canvas
- Style: hand-drawn, rainbow chaos, doodle elements
- See: `../Design/STITCH-PROMPT-WREN-IDLE.md` for full design specs

## Deploy

Push to `main` branch → Cloudflare Pages auto-deploy.
